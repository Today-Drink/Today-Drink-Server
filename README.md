# 💻 Today-Drink-Server
<p>오늘한잔 프로젝트 백엔드 파트입니다.</p>


# 🔧 개발환경
<h3>Framework / Library</h3>
<p>- SpringBoot 2.7.9</p>
<p>- Spring Data JPA</p>
<p>- Query DSL</p>
<p>- Spring Security</p>

<h3>Build Tool</h3>
<p>- Gradle</p>

<h3>Database</h3>
<p>- Postgresql </p>
<p>- AWS RDS</p>

<h3>Deployment</h3>
<p>- AWS EC2</p>
<p>- Docker</p>
<p>- Jenkins</p>
